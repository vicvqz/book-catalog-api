# Book Catalog API

API REST desarrollada con Node.js y Express para gestionar un catálogo de libros.

## Endpoints

### Obtener libros

GET /books

### Crear libro

POST /books

### Eliminar libro

DELETE /books/:id

## Ejecutar localmente

Instalar dependencias:

```bash
npm install
```

Ejecutar aplicación:

```bash
npm start
```

Ejecutar pruebas:

```bash
npm test
```
